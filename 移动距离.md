### 题目描述
X星球居民小区的楼房全是一样的，并且按矩阵样式排列。

其楼房的编号为 $1,2,3…$

当排满一行时，从下一行相邻的楼往反方向排号。

比如：当小区排号宽度为 $6$ 时，开始情形如下：

```
1  2  3  4  5  6
12 11 10 9  8  7
13 14 15 .....
```

我们的问题是：已知了两个楼号 $m$ 和 $n$，需要求出它们之间的最短移动距离（不能斜线方向移动）。

#### 输入格式

输入共一行，包含三个整数 $w,m,n$，$w$ 为排号宽度，$m,n$ 为待计算的楼号。

#### 输出格式

输出一个整数，表示 $m, n$ 两楼间最短移动距离。

#### 数据范围

$1 \le w,m,n \le 10000$,

#### 输入样例：

```
6 8 2
```

#### 输出样例：

```
4
```

### 算法
模拟

#### C++ 代码
```
#include <bits/stdc++.h>
using namespace std;
int w,m,n;
int line1,line2;
int pos1,pos2;
int res;
int main () {
    cin>>w>>m>>n;
 
if (m % w == 0) {
    line1 = m / w;
} else {
    line1 = m / w + 1;
}


if (n % w == 0) {
    line2 = n / w;
} else {
    line2 = n / w + 1;
}
    if(line1%2==0&&m%w!=0){
        pos1=w-m%w+1;
    }else if(line1%2==0&&m%w==0){
        pos1=1;
    }else if(line1%2!=0&&m%w!=0){
        pos1=m%w;
    }else if(line1%2!=0&&m%w==0){
        pos1=w;
    }
        
    
    if(line2%2==0&&n%w!=0){
        pos2=w-n%w+1;
    }else if(line2%2==0&&n%w==0){
        pos2=1;
    }else if(line2%2!=0&&n%w!=0){
        pos2=n%w;
    }else if(line2%2!=0&&n%w==0){
        pos2=w;
    }
    res=abs(pos2-pos1)+abs(line2-line1);
    cout<<res<<endl;
    return 0;
}
```
