### 题目描述
你现在被困在一个三维地牢中，需要找到最快脱离的出路！

地牢由若干个单位立方体组成，其中部分不含岩石障碍可以直接通过，部分包含岩石障碍无法通过。

向北，向南，向东，向西，向上或向下移动一个单元距离均需要一分钟。

你不能沿对角线移动，迷宫边界都是坚硬的岩石，你不能走出边界范围。

请问，你有可能逃脱吗？

如果可以，需要多长时间？

#### 输入格式

输入包含多组测试数据。

每组数据第一行包含三个整数 $L,R,C$ 分别表示地牢层数，以及每一层地牢的行数和列数。

接下来是 $L$ 个 $R$ 行 $C$ 列的字符矩阵，用来表示每一层地牢的具体状况。

每个字符用来描述一个地牢单元的具体状况。

其中, 充满岩石障碍的单元格用”#”表示，不含障碍的空单元格用”.”表示，你的起始位置用”S”表示，终点用”E”表示。

每一个字符矩阵后面都会包含一个空行。

当输入一行为”0 0 0”时，表示输入终止。

#### 输出格式

每组数据输出一个结果，每个结果占一行。

如果能够逃脱地牢，则输出”Escaped in x minute(s).”，其中X为逃脱所需最短时间。

如果不能逃脱地牢，则输出”Trapped!”。

#### 数据范围

$1 \le L,R,C \le 100$

#### 输入样例：

```
3 4 5
S....
.###.
.##..
###.#

#####
#####
##.##
##...

#####
#####
#.###
####E

1 3 3
S##
#E#
###

0 0 0
```

#### 输出样例：

```
Escaped in 11 minute(s).
Trapped!
```

---
### 算法
bfs

#### C++ 代码
```
#include <bits/stdc++.h>
using namespace std;

const int N=110;
char n[N][N][N];
bool st[N][N][N];
int m[N][N][N];
int l,r,c;
int dx[6] = {1, -1, 0, 0, 0, 0};
int dy[6] = {0, 0, 1, -1, 0, 0};
int dz[6] = {0, 0, 0, 0, 1, -1};

typedef pair<int ,pair<int,int> > PII;
PII startt,endd;
queue<PII> q;

int bfs(PII a,PII b){
    memset(st,0,sizeof(st));
    memset(m,0,sizeof(m));
    while (!q.empty()) {
    q.pop();
}
    q.push(a);
    while(!q.empty()){
        auto t=q.front();
        q.pop();
                for(int k=0;k<6;k++){
                    int x=t.first+dx[k];
                    int y=t.second.first+dy[k];
                    int z=t.second.second+dz[k];
                    if(x<1||x>l||y<1||y>r||z<1||z>c)continue;
                    if(n[x][y][z]=='#')continue;
                    if(st[x][y][z])continue;
                    st[x][y][z]=true;
                    m[x][y][z]=m[t.first][t.second.first][t.second.second]+1;
                    if(n[x][y][z]=='E')return m[x][y][z];
                    q.push({x,{y,z}});
                }
    }
    return 0;
}

int main () {
    
    while(cin>>l>>r>>c&&l!=0&&r!=0&&c!=0){
    for(int i=1;i<=l;i++){
        for(int j=1;j<=r;j++){
            for(int k=1;k<=c;k++){
                cin >> n[i][j][k];
            }
        }
    }
    for(int i=1;i<=l;i++){
        for(int j=1;j<=r;j++){
            for(int k=1;k<=c;k++){
                if(n[i][j][k]=='S')startt={i,{j,k}};
                if(n[i][j][k]=='E')endd={i,{j,k}};
            }
        }
    }
    int back_value=bfs(startt,endd);
    if(back_value!=0)printf("Escaped in %d minute(s).\n",back_value);
    else printf("Trapped!\n");
    }
    return 0;
    
}
```
#### 反思
这道题还是有挺多的细节的，首先就是c++里面的queue我还是不太熟悉，还得熟悉一下，然后就是每个数组的初始化，因为这道题会输入很多个数据，所以在每一个数据之前都应该给每个数组初始化，这里使用了三个数组，一个数组是用来读入输入的数据，一个bool数组是用来判断现在这个地方有没有走过，还有一个int 类型的数组是为了存一下从起点的位置坐过来需要多少步，这些应该就是这类bfs的题目的标准的做法，还有就是需要注意一下不同行代码的一个先后顺序，这一点很重要，还有就是这里读入最后使用cin来读入，因为scanf读入每一行结尾的那个\n可能会有影响，然后就是这个偏移量的做法，非常的巧妙，在很多的题目中都有用到，这种做法应该记住一下。还有就是有地方的判断的条件需要注意一下，因为存入的数组下标可能不是从0开始的，所以需要注意一下判断的条件
