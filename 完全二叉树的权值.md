### 题目描述
给定一棵包含 $N$ 个节点的完全二叉树，树上每个节点都有一个权值，按从上到下、从左到右的顺序依次是 $A_1, A_2, · · · A_N$，如下图所示：

![QQ截图20191205124611.png](https://cdn.acwing.com/media/article/image/2019/12/05/19_2f0cae5817-QQ截图20191205124611.png)

现在小明要把相同深度的节点的权值加在一起，他想知道哪个深度的节点权值之和最大？

如果有多个深度的权值和同为最大，请你输出其中最小的深度。

注：根的深度是 $1$。

#### 输入格式

第一行包含一个整数 $N$。

第二行包含 $N$ 个整数 $A_1, A_2, · · · A_N$。

#### 输出格式

输出一个整数代表答案。

#### 数据范围

$1 \le N \le 10^5$,  
$-10^5 \le A_i \le 10^5$

#### 输入样例：

```
7
1 6 5 4 3 2 1
```

#### 输出样例：

```
2
```

---
### 算法
双指针

#### C++ 代码
```
#include <bits/stdc++.h>
using namespace std;

const int N =1e5+10;

int a[N];
int m;
long long cnt;
int level;
int main () {
    int n;
    cin>>n;
    for(int i=1;i<=n;i++)scanf("%d",&a[i]);
    int q=0;
    int p=0;
    int m=0;
    long long maxx=INT_MIN;
    for(q=0;m<=n;q++){
        p=pow(2,q);
        int i=m+1,j=m+1;
        m+=p;
        if(i>n&&j>n)break;
      
        
        for(;j-i<pow(2,q)&&j<=n;j++){
            cnt+=a[j];
        }
        if(cnt>maxx){
            level=q+1;
            maxx=cnt;
        }
        cnt=0;
    }
    cout<<level;
    return 0;
}
```
#### 反思
写了这道题我才知道原来完全二叉树最后一层并不是全满的，所以循环的条件要加上j<=n，来避免最后一层不满的时对结果造成的影响，还有就是题目中没有说所有的数据都是正数，所以说是存在一种情况，最大值还要小于零的，所以初始化maxx的时候使用INT_MIN，让maxx最初的时候是最小的，然后还是老生常谈的问题，需要注意int的最大值，这道题因为数据可能很大，所以cnt和maxx都要使用long long 来存
