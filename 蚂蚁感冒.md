### 题目描述
长 $100$ 厘米的细长直杆子上有 $n$ 只蚂蚁。

它们的头有的朝左，有的朝右。

每只蚂蚁都只能沿着杆子向前爬，速度是 $1$ 厘米/秒。

当两只蚂蚁碰面时，它们会同时掉头往相反的方向爬行。

这些蚂蚁中，有 $1$ 只蚂蚁感冒了。

并且在和其它蚂蚁碰面时，会把感冒传染给碰到的蚂蚁。

请你计算，当所有蚂蚁都爬离杆子时，有多少只蚂蚁患上了感冒。

#### 输入格式

第一行输入一个整数 $n$, 表示蚂蚁的总数。

接着的一行是 $n$ 个用空格分开的整数 $X_i$, $X_i$ 的绝对值表示蚂蚁离开杆子左边端点的距离。

正值表示头朝右，负值表示头朝左，数据中不会出现 $0$ 值，也不会出现两只蚂蚁占用同一位置。

其中，第一个数据代表的蚂蚁感冒了。

#### 输出格式

输出1个整数，表示最后感冒蚂蚁的数目。

#### 数据范围

$1 < n < 50$,  
$0 < |X_i| < 100$

#### 输入样例1：

```
3
5 -2 8
```

#### 输出样例1：

```
1
```

#### 输入样例2：

```
5
-10 8 -20 12 25
```

#### 输出样例2：

```
3
```

### 算法
数学

#### C++ 代码
```
#include <bits/stdc++.h>
using namespace std;
const int N =60;
int a[N];
int main () {
    int n;
    cin>> n;
    for(int i=0;i<n;i++)cin>>a[i];
    int left=0;
    int right=0;
    for(int i=1;i<n;i++){
        if((a[i]>0&&a[0]>0&&a[i]<a[0])||(a[i]>0&&a[0]<0&&a[i]+a[0]<0))left++;
        if((a[i]<0&&a[0]>0&&a[i]+a[0]<0)||(a[i]<0&&a[0]<0&&a[i]<a[0]))right++;
        }
    if(a[0]<0&&left!=0){
        cout<< left +right +1;}
    else if(a[0]>0&&right!=0){cout<< left+right +1;}
    else {cout << "1";}
    return 0;
    
}
```
#### 反思
这道题最重要的一个地方是需要想到，掉头等于穿过，因为相碰之后两两个蚂蚁的状态是完全一致的，所以完全可以看成是两个蚂蚁直接穿过，还有就是这道题判断感冒的蚂蚁左右的蚂蚁的时候可以使用c++的标准库里面的abs（）这个函数，表示加上绝对值，方便比较两个数绝对值的大小（但是我当时写的时候不知道，导致写了一个非常抽象的判断条件）
